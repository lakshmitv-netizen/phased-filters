Section 07

Cell Interactions

How users interact with grid cells

👆 Single Click

  • Select individual cell
  • Shows blue selection border
  • Opens edit info popover if cell has history

⇧ Shift + Click

  • Select range of cells
  • From last selected to current
  • Adds to existing selection

⌘ Cmd/Ctrl + Click

  • Toggle cell in selection
  • Add or remove individual cells
  • Build non-contiguous selection

👆👆 Double Click

  • Enter edit mode
  • Type new value
  • Press Enter to save, Escape to cancel

Context Menu (Right-Click)

📋 Copy & Paste

  • Copy cell value to clipboard
  • Paste value from clipboard

🔒 Lock / Unlock

  • Prevent cell from being edited
  • Excluded from value propagation

Section 08

Value Propagation

Automatic recalculation of parent/child values

⬆️ Upward Propagation

When you edit a child cell, parent totals automatically recalculate

Parent: 180 → 200
Auto-updated
↑
Child A
100
Child B
80 → 100
User edit

⬇️ Downward Propagation

When you edit a parent cell, children are proportionally adjusted

Parent: 200 → 300
User edit
↓
Child A
100 → 150
Proportional
Child B
100 → 150
Proportional

🔒 Locked Cells

Locked cells are excluded from value propagation. When distributing values downward, locked cells maintain their value and the remaining amount is distributed among unlocked children.
